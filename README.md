## 🌱 Project Overview

**Plant-Pal** is a responsive plant care tracking website built with **React** for the client-side. It allows users to:

- Track their plants and their care routines.
- Add new plants with care information.
- Update plant details and monitor plant health.
- View new and popular plants with beautiful UI.
- Engage with a community of plant lovers.
- Secure authentication and registration system.

This project focuses on **user-friendly UI**, **dynamic interactivity**, and **real-time updates** for a seamless plant tracking experience.

## 🛠️ Technology Stack (Client-Side)

- **React** – Frontend library for building UI
- **React Router DOM** – Routing between pages
- **React Icons** – Icon integration
- **Tailwind CSS** – Styling and responsive design
- **DaisyUI** – UI components for React
- **React Lottie** – Animated backgrounds
- **SweetAlert2** – Alerts for user actions
- **React Toastify** – Toast notifications
- **Fetch API / Axios** – Interacting with server REST API
## 🚀 Getting Started (Client-Side)

### Prerequisites

- Node.js installed
- npm or yarn package manager
- Backend API running (Server-Side)
