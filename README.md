## ğŸŒ± Project Overview

**Plant-Pal** is a responsive plant care tracking website built with **React** for the client-side. It allows users to:

- Track their plants and their care routines.
- Add new plants with care information.
- Update plant details and monitor plant health.
- View new and popular plants with beautiful UI.
- Engage with a community of plant lovers.
- Secure authentication and registration system.

This project focuses on **user-friendly UI**, **dynamic interactivity**, and **real-time updates** for a seamless plant tracking experience.
